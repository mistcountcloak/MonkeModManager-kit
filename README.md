## Overview
**Monke Mod Manager** is a clean, fast, and user-friendly mod manager for **Gorilla Tag**.
It simplifies the process of installing, managing, and updating mods, allowing you to easily customize your Gorilla Tag experience.

<img width="568" height="408" alt="image" src="https://github.com/user-attachments/assets/e9486363-24dc-48cb-829a-af93d7994852" />




## Key Features
- One-click mod installation
- Large built-in mod library
- Automatic mod updates
- Custom room support
- Easy mod enabling/disabling
- Clean and modern interface
- Backup & restore functionality
- Low resource usage

## How to Use
1. Download the latest release
2. Extract the archive
3. Run `MonkeModManager.exe` as Administrator
4. Select your mods and install
5. Launch Gorilla Tag and enjoy!
